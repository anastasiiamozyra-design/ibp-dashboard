# IBP-Style Planning Dashboard

## Overview
This project is a Streamlit-based mini planning tool that simulates IBP-style demand, inventory, and replenishment logic.

## What it demonstrates
Users can adjust key planning inputs such as:
- average daily demand
- demand variability
- lead time
- initial inventory
- safety stock factor

The dashboard then compares:
- a basic replenishment rule
- an IBP-style planning logic

## KPIs
The tool shows the impact on:
- service level
- stockouts
- average inventory

## Why it matters
This project demonstrates how planning tool logic can be translated into a simple, interactive MVP that supports supply chain decision-making.
